# 🛡️ Cybersecurity Chatbot – PROG6221 Part 1

A C# console-based chatbot that educates users on cybersecurity topics. The bot features voice greetings, ASCII art, personalized responses, and a clean console UI. This project is built with modular C# code, version-controlled via Git, and includes a CI pipeline using GitHub Actions.

---

## 🚀 Features

- 🎤 **Voice Greeting** – Plays a `.wav` audio greeting when the program starts.
- 🖼️ **ASCII Art** – Displays a cybersecurity-themed logo using ASCII characters.
- 💬 **User Interaction** – Asks for the user's name and personalizes responses.
- 🧠 **Cybersecurity Q&A** – Responds to questions about passwords, phishing, and more.
- ✅ **Input Validation** – Handles empty or unrecognized inputs gracefully.
- 🎨 **Enhanced Console UI** – Colored text, borders, and typing effects for a polished look.
- 🧱 **Modular Code Structure** – Separated into `Chatbot.cs`, `User.cs`, and `AudioPlayer.cs`.
- 🔁 **Git & CI** – Minimum 6 commits, public GitHub repo, and GitHub Actions CI with a green build badge.

## 🛠️ Technologies & Tools

- **Language:** C# (.NET Console App)
- **Audio:** `System.Media.SoundPlayer`
- **ASCII Art:** Generated via [patorjk.com](https://patorjk.com/software/taag/)
- **Voice Recording:** Windows Voice Recorder or [TTSM3](https://ttsm3.com)
- **Version Control:** Git + GitHub
- **CI/CD:** GitHub Actions (.NET workflow)
